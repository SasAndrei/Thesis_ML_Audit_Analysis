Intro to deep learning for medical imaging by MD.ai Lesson 2.

Lung X-Rays Semantic Segmentation This lesson applies a U-Net for Semantic Segmentation of the lung fields on chest x-rays. The MD.ai annotator is used to view the DICOM images, and to create the image level annotation. The MD.ai python client library is then used to download images and annotations, prepare the datasets, then are then used to train the model for classification.

  • MD.ai annotator project URL: https://public.md.ai/annotator/project/aGq4k6NW
  • MD.ai python client libray URL: https://github.com/mdai/mdai-client-py
